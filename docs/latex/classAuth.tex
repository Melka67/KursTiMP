\doxysubsection{Подробное описание}
Класс для аутентификации клиентов 

Обеспечивает проверку учетных данных клиентов, используя механизм соли и хэширования SHA-\/224.

\begin{DoxyNote}{Заметки}
Все методы статические -\/ не требует создания экземпляра 
\end{DoxyNote}

Выполняет полный процесс аутентификации клиента

Протокол аутентификации\+:
\begin{DoxyEnumerate}
\item Клиент отправляет\+: LOGIN(4) + SALT(16 hex) + HASH(56 hex)
\item Сервер проверяет формат
\item Сервер вычисляет и сравнивает хэш
\item Сервер отправляет OK или ERR
\end{DoxyEnumerate}

\hline
\end{DoxyParams}

true Клиент успешно аутентифицирован 

false Ошибка аутентификации
\end{DoxyReturn}
Процесс аутентификации\+:
\begin{DoxyEnumerate}
\item Получение данных от клиента (76 байт)
\item Разбор на компоненты\+: логин, соль, хэш
\item Валидация формата
\item Проверка учетных данных
\item Отправка результата клиенту
\end{DoxyEnumerate}

\begin{DoxyNote}{Заметки}

\end{DoxyNote}
