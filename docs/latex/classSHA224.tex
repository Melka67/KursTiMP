\doxysubsection{Подробное описание}
Класс для вычисления SHA-\/224 хэшей 

Предоставляет статические методы для\+:
\begin{DoxyItemize}
\item Вычисления хэшей от произвольных данных
\item Вычисления хэшей с солью
\item Преобразования в hex строки
\item Валидации hex строк
\end{DoxyItemize}

\begin{DoxyNote}{Заметки}
Использует Open\+SSL EVP API (рекомендовано для Open\+SSL 3.\+0+) 
\end{DoxyNote}
