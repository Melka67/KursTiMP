Класс сервера для обработки клиентских подключений 

Отвечает за\+:
\begin{DoxyItemize}
\item Создание и настройку сокета
\item Ожидание входящих подключений

Запускает сервер и начинает прослушивание порта


\begin{DoxyParams}{Аргументы}

true Сервер успешно запущен 

false Ошибка запуска сервера
\end{DoxyReturn}
Последовательность действий\+:
\begin{DoxyEnumerate}
\item Загрузка базы данных
\item Создание сокета
\item Привязка к порту
\item Начало прослушивания
\item Обработка входящих подключений
\end{DoxyEnumerate}


\begin{DoxyParams}{Аргументы}

\hline
\end{DoxyParams}

true Сервер успешно запущен 

false Ошибка при запуске сервера
\end{DoxyReturn}
Метод выполняет полную инициализацию сервера\+:
\begin{DoxyEnumerate}
\item Загружает базу данных пользователей
\item Создает сокет сервера
\item Настраивает и привязывает сокет
\item Начинает прослушивание порта
\item Входит в бесконечный цикл обработки клиентов
\end{DoxyEnumerate}
